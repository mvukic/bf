Usage:

```shell
 java bf-interpreter.jar <path-to-program> [OPTIONS]
```

Options

| Required | Key           | Value                | Example                   |
|----------|---------------|----------------------|---------------------------|
| No       | --init-memory | Initial memory value | `--init-memory 1,2,3,4,5` |
| No       | --init-input  | Input values         | `--init-input 1,2,3,4,5`  |
|          |               |                      |                           |
